# Manual Scripts

This directory collects historical ad-hoc scripts that were previously stored in the repository root. They are kept for reference when debugging publisher flows or re-running bespoke experiments, but they are **not** part of the supported CLI surface.

Scripts are grouped by prefix (e.g. `siam_`, `wiley_`, `vpn_`) to make discovery easier. When adding new throwaway experiments, drop them here instead of the repository root so the top-level layout stays clean.
